# Workflow Management System – Figma Design

## Project Overview
This project contains the UI/UX design of a **Workflow Management System** created using Figma.  
The design demonstrates how workflows, tasks, and organizational roles interact within a system.

The prototype allows users to navigate through the screens and understand the workflow process visually.
